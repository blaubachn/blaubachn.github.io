Eye of Gnome Remote

About

This is a very customized project written to sit on top of either the Eye of Gnome or Eye of Mate slideshow feature. It stores a file directory and starts or stops a slideshow of the pictures at that location via a Cherry Py web server.

The purpose is being able to have a computer that is used as an announcements slideshow presenter that can be controlled without a mouse or keyboard. Because Eye of Gnome and Eye of Mate are the core of the functionality, this is a Linux only project.

If you run into any issues or have any ideas that you think would make the remote better, please send me an email at nathanblaubach@gmail.com

Development

This app was created using Python, CherryPy, Eye of Gnome and Eye of Mate.
